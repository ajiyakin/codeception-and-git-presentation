Pengetahuan dasar tentang Git

  • Clone - Unduh projek ke lokal
  • Remote - Alamat server project
  • Branch - Cabang (tempat) penyimpanan sementara
  • Pull - Ambil perubahan terbaru dan satukan dengan keadaan file sekarang
  • Fetch - Ambil perubahan terbaru dan satukan dengan keadaan file pada branch tertentu
  • Commit - Simpan keadaan file sekarang pada branch lokal
  • Push - Update server dengan keadaan file lokal saat ini

File-file Penting di Git

  • .gitignore - Acuhkan file yang terdaftar
  • .gitkeep - BUKAN fitur dari Git, biasa digunakan untuk merekam folder kosong
  • .git/hooks/ - Script (bash) yang dijalankan setelah/sebelum operasi Git
